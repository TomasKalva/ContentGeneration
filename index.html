Getting Started

Unity Application - NoesisGUI

Visual Studio and Blend

The generated application project is ready to run within both Visual Studio and Blend for Visual Studio. To run the project open the .csproj file with Visual Studio or Blend and press F5.

Note that XAMLs and Code-Behind classes are shared with Unity.

Unity

The generated Unity project is located in the Assets folder. Please follow these steps:

  1. Launch Unity and open the generated project.
  2. Download latest version of NoesisGUI and import it into the project.
  3. Open the scene located in the root folder.
  4. Press play in the editor.
